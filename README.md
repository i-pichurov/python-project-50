# Project description

The project is called Difference Ð¡alculator. It takes two .json or .yml files and shows their differences. When starting a project, you can choose which of the three formats the result will be given in:
- stylish;
- plain;
- json.

# Installing

Copy the contents of the repository:
    
    git clone git@github.com:i-pichurov/python-project-50.git

Install the package using the command in the Makefile:
    
    make package-install

After installing the package, you can get help information on the utility with the following command:

    gendiff -h
